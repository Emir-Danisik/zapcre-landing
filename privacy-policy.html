Privacy Policy for Zapcre

Last Updated: June 30, 2025

Effective Date: June 30, 2025

Website: https://zapcre.com

1. Introduction

Zapcre ("we", "us", "our") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, store, and share information when you use Zapcre, particularly in relation to your Google account and Gmail data. This policy is designed to comply with Google’s OAuth 2.0 verification, Gmail Restricted Scopes policy, Google API Services User Data Policy—including the Limited Use requirements—and broader global privacy regulations (GDPR, CCPA, etc.).

By using Zapcre and authorizing access to your Google account, you consent to the data practices described in this policy.

2. Scope of Application

This Privacy Policy applies to all users of Zapcre, particularly those using Gmail integrations via Google OAuth 2.0. The policy covers:

Data obtained through Gmail API restricted/sensitive scopes

Google OAuth profile data

App interactions involving email reading, metadata parsing, and workflow automations

Zapcre does not use your Gmail data for advertising, user profiling, or any use unrelated to explicitly described functionality.

3. Data Collected

3.1 Google OAuth Profile Data

If you authorize Google Sign-In, we may access:

Name

Email address

Profile image

Google User ID

This is used to personalize your experience and identify your account within Zapcre.

3.2 Gmail-Specific Data

If you authorize Gmail scopes, we may access:

Email metadata: sender, recipient, subject, timestamps

Email body (if explicitly required by feature and consented)

Attachments (only when necessary for file extraction features)

We only request scopes necessary to deliver the functionality you select, including:

https://www.googleapis.com/auth/gmail.readonly

https://www.googleapis.com/auth/gmail.modify

https://www.googleapis.com/auth/gmail.send

https://mail.google.com/ (Restricted Scope)

We explicitly avoid requesting full access unless required for the selected user functionality.

4. Use of Data

Zapcre uses your data exclusively for the following purposes:

Automating email-based real estate workflows

Extracting lease data, attachments, and contact info from emails

Allowing users to send emails through Gmail via Zapcre UI

Organizing and visualizing email data through dashboards

Enhancing productivity through email tagging or labeling

Zapcre does not:

Sell or rent your data to third parties

Use data for advertising or marketing

Access, use, or share your Gmail content beyond your explicit use case

5. Limited Use Disclosure

Zapcre’s use of data obtained via Gmail API fully complies with Google’s Limited Use Policy, which includes:

No human review of Gmail content unless explicitly requested by the user for support

No use of Gmail data for ads or marketing

No use of data for user profiling or personalization beyond the app

We never share Gmail data with external vendors or partners unless required by law.

6. Data Storage and Retention

Zapcre does not store Gmail data on our servers unless a specific feature requires temporary caching (e.g., previewing a message). If so, data is:

Encrypted at rest using AES-256

Auto-deleted within 30 minutes or upon logout

Never included in analytics, logs, or backups

If you disconnect your Google account, all associated Gmail content is purged immediately.

7. Data Deletion and User Control

You may:

Revoke access at any time via Google Account Permissions (https://myaccount.google.com/permissions)

Email us at [email protected] to delete your entire Zapcre account and associated data

Request access or correction of stored data under GDPR or CCPA

Export your data via a simple JSON request (we'll email it to you within 7 days)

8. Security Measures

We implement industry-standard security protocols including:

OAuth 2.0 for authentication (no password collection)

HTTPS/TLS 1.2+ for all data in transit

Client-side encryption where applicable

IP-based rate limiting and suspicious session detection

Zapcre has passed internal security reviews and complies with Google’s data handling rules for apps with sensitive and restricted Gmail scopes.

9. Children’s Privacy

Zapcre is not intended for children under 13. We do not knowingly collect personal data from users under 13. If we discover such data was collected, it will be deleted immediately.

10. Disclosure of Information

We may disclose personal data only:

To comply with legal obligations (e.g., subpoena, court order)

To protect the safety and rights of users or the public

To third-party auditors during official security audits (e.g., CASA Tier 2 if using restricted scopes)

11. Third-Party Services

Zapcre may use the following third-party services:

Google Cloud – OAuth, Gmail API access

Vercel – Frontend hosting

Firebase (Google) – Authentication, hosting, and cloud functions

Sentry – Error logging (anonymous, no email/Gmail content)

Clerk.dev or Supabase (optional) – User authentication layer

Each provider is fully GDPR/CCPA compliant. We never share Gmail content with any provider.
